
# sample application

This application illustrates some of the possible functions that can be performed using the REST API.
These include:

- Authorize
- list items
- list orders
- to write an article
- Query article by External_Key
- Clearing the Cache(OutputCache, EF Cache etc.)
- List branches
- list customers
- Call up order management
- availabilities
- Create order
- List Coupons
- availabilities
- position updates
